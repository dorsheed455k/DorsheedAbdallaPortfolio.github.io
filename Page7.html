Arduino Ultrasonic Sensor + Sensor Track Java Swing Interface

A project that intergrates Arduino digital interactivity with Java graphical interface animation with the use of the ultrasonic sensor device to receive readings on the distance of an object right in front of it. This can be used to control the animation panel in the interface.
