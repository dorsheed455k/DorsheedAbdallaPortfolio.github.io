Password Generator

Functioning password generation application making use of the JavaFx framework, providing users the functionality of choosing the field range of generated passwords

Features

  • Length; users can adjust the length of the password ranging from 5 to 20 characters
  • Uppercase; users can check uppercase characters (A-Z) in the field of characters in password
  • Lowercase; users can check lowercase characters (a-z) in the field of characters in password
  • Numbers; users can check numerical (0-9) characters in the field of characters in password
  • Symbols; users can check symbols or other characters in the field of characters in password
  • Generate; users press button once selected field paramters for password to generate passwords in field for unlimited times
  • Copy; users press button once desired password is generated and chooses to paste it elsewhere
  • Clear; users press button to create blank password field
  • Copy to File; users press button to write text in password field to .txt file

Passwords saved to text file upon pressing copy to file button
